# doorlock-espnow
This is an example of remotely unlocking a door lock using ESPNOW!

![496-3_bb](https://user-images.githubusercontent.com/106683637/175772610-b54e7e69-28da-4d6e-a8f8-84ba819c137b.jpg)

Below is a video of the final result! Sorry, it's in Korean!

[![Video Label](http://img.youtube.com/vi/OhPPtS6Nshs/0.jpg)](https://youtu.be/OhPPtS6Nshs?t=9671)

1. The door lock and wemos d1 r1 are connected using a relay!
2. Connect the button to the remote control board!
3. Press the button on the remote control board to unlock the door lock!
4. The LED turns on depending on the success or failure of the command transmission!
